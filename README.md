# Chris G Pinto Portfolio Website

Welcome to the source code for my personal portfolio website, **chrispinto.tech**. This website showcases my background, education, professional experience, technical projects, skills, achievements, and contact information in a clean and responsive design.

The portfolio serves as a central hub for recruiters, employers, and collaborators to learn more about me and explore my work.

---

## 🚀 Features

* 📖 About Me section introducing who I am
* 🎓 Education section highlighting my academic background
* 💼 Experience section showcasing professional and extracurricular experience
* 💻 Projects section featuring my software development projects
* 🛠️ Skills & Achievements section displaying technical skills and accomplishments
* 📬 Contact section with ways to get in touch
* 📱 Fully responsive layout for desktop, tablet, and mobile devices
* ⚡ Fast-loading static website built using modern web technologies

---

## 🛠️ Built With

* **HTML** (64.8%)
* **CSS** (27.1%)

## 📦 Installation

Clone this repository:

```bash
git clone https://github.com/cpint2410/Phase1-WebTech.git
```

Navigate to the project directory:

```bash
cd Phase1-WebTech
```

Open the website by launching the `index.html` file in your preferred web browser.

Alternatively, if you have Visual Studio Code installed, you can use the **Live Server** extension for a local development server.
